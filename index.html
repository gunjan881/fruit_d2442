Fruit Suggestion Chatbot (Watson Embed)

A minimal HTML page that embeds the IBM Watson Assistant web chat.

How to use
  1. Save this file as fruit_chatbot_watson.html.
  2. Open it in a browser (preferably served over https if your Watson integration requires it).
  3. The Watson chat widget will automatically render using the integration credentials below.

Integration snippet included:

<script> window.watsonAssistantChatOptions = { integrationID: "566d834c-cebc-460f-89e0-6bbb2644e76a", region: "eu-de", serviceInstanceID: "2492287c-8841-446b-8370-8654730a668e", onLoad: async (instance) => { await instance.render(); } }; setTimeout(function(){ const t=document.createElement('script'); t.src="https://web-chat.global.assistant.watson.appdomain.cloud/versions/" + (window.watsonAssistantChatOptions.clientVersion || 'latest') + "/WatsonAssistantChatEntry.js"; document.head.appendChild(t); }); </script>
